基本URL路由（仅供参考）

商城首页：/

商品信息：

/product/10000100001（显示商品信息）

/product/newproduct（发布新的商品信息）

/product/modify/1000010001（修改商品信息）

店铺展示：/shop/111（店铺编号）

搜索结果：/searchresult/keyword=（搜索关键字）

认证页面：

/auth/login（登陆）

/auth/register（注册）

/auth/shopapply（店铺申请）



卖家账户：

/account/saler/id（卖家账户信息）



管理员界面：

/admin/unverifyshop（待审核店铺）

/admin/verifiedshop（审核完成的店铺）
